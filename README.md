# Stock Market Charting
这是一个实训项目，它实现了通过多个图表方式来展现股标价格的变化趋势，以供用户在买卖股票时提供决策依据。主要图表如下：
1，一个公司的股票价格走势；
2，一段时间内不同公司的股票价格对比图；
3，一段时间内不同行业的股票价格对比图；
。。。。。。
